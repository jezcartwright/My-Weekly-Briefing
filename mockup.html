How this works

  • JS measures the Leadership pill's horizontal centre on load, sets that as the "axis" via a CSS variable
  • The category label uses position:relative; left:axis-x; transform:translateX(-50%) so its centre always lands on that axis
  • The label is on the same vertical line as the LEADERSHIP pill above, regardless of which category is currently active
  • Dashed guideline is just for this mockup — would be invisible on the live site

What you'd see on the live site

Same behaviour, no visible guideline. The label sits below the Leadership pill's centre. Tap Markets → band turns blue, label says MARKETS, but MARKETS is centred on the same x-position. Tap Technology → band turns teal, label says TECHNOLOGY, still on the same line.
