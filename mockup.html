The final spec (locked in)

Compression: C1 — proportional ~25% shrink. Masthead ~155px on phone vs ~200px before. Same centred-stack layout, just smaller type and tighter spacing.

Menu font: M3 — Inter small caps. 11px, weight 700, letter-spacing 1.4px, uppercase. Matches the tab labels typographically.

Aa fix: "Text size" icon has its own .ico-aa class with text-transform:none, so mixed-case "Aa" is preserved despite the parent's uppercase transform.

Avatar (clickable): opens account menu with user name + email header, then Account / Subscription / Sign out items.

Hamburger (clickable): opens app menu with PDF / My Notes / Liked / Library / Text size.

Tabs: centred with text-align wrapper + inline-flex content. Scrolls naturally on phone (where six tabs overflow); centres without clipping on iPad+.
