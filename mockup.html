How they differ

A · Box-to-box: The container of the label aligns with the container of the pill. Mechanically tidy but the alignment isn't visually obvious — the pill text is inset 13.5px inside its own box, so the words "LEADERSHIP" in the pill and "LEADERSHIP" in the band still don't sit under each other.

B · Text-to-text: The actual letters line up. Looking down the column you see L–L, E–E, A–A. This is the alignment relationship that the eye naturally reads as "the same position".

Both are static — once set, the label never moves when you switch tabs. Only the band recolours and the label text changes (LEADERSHIP → MARKETS → PSYCHOLOGY → etc.) but the horizontal position stays put.

Implementation is identical effort — single CSS value change to .ci padding-left.
