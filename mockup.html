A few things worth knowing as you decide

  • Mobile width. All four options are shown at iPhone width (~375px). On iPad/desktop the masthead gets wider and the toolbar fills less of the row — Option A's toolbar would feel less crowded there. Option C looks identical on every width.
  • The π logo position. Set bottom-left, aligned with the wordmark baseline. The "Performance Intelligence" eyebrow stacks on two lines and sits above "Weekly Briefing", to the right of the π.
  • The date-and-avatar pill. Now one unified white capsule containing both — date text + circular avatar inside. Replaces the old "Tuesday, 2 June 2026" + "Jez" floating elements. Tappable as a single unit; tap → account menu.
  • What the toolbar buttons currently do (for reference when deciding): PDF = export current view; Notes = side panel of all your notes; Liked = saved/favourited topics; Library = full archive of past briefings; Font-size = reader text size.
  • Mike will see the same banner. Whatever's chosen is global; we can't personalise per user without extra plumbing.
